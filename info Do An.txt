Dataset
Context 
These files contain metadata for all 65,000 movies listed in. Full MovieLens Dataset. The dataset consists of movies released on or before July 2017. Data points include cast, crew, plot keywords, budget, revenue, posters, release dates, languages, produMion companies, countries, TMDEI vote counts and vote averages. This dataset also has files containing 26 million ratings from 270,000 users for all 45,000 movies. Ratings are on a scale 00,5 and have been obtained from the official GroupLem vvebsite
